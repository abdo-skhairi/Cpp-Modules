## 🧠 Overview

This repository contains my implementation of the **C++ Modules (00 to 04)** from the 42 curriculum.

These modules introduce the core concepts of **Object-Oriented Programming (OOP)** in C++, starting from basic syntax and progressing to **inheritance** and **polymorphism**.

---

## 🧩 Modules

### 🔹 Module 00 — Basics of C++

**Concepts**
- Namespaces  
- Classes & member functions  
- Input / Output streams  
- Static vs instance attributes  

**Exercises**
- `megaphone`
- `phonebook`

---

### 🔹 Module 01 — Memory & References

**Concepts**
- Pointers & references  
- Dynamic memory allocation (`new` / `delete`)  
- File streams  

**Exercises**
- `zombie`
- `brain`
- `replace`

---

### 🔹 Module 02 — Ad-hoc Polymorphism

**Concepts**
- Operator overloading  
- Orthodox Canonical Form  
- Fixed-point numbers  

**Exercises**
- `Fixed`
- `Point`
- `BSP`

---

### 🔹 Module 03 — Inheritance

**Concepts**
- Inheritance  
- Base and derived classes  
- Constructors and destructors  

**Exercises**
- `ClapTrap`
- `ScavTrap`
- `FragTrap`

---

### 🔹 Module 04 — Polymorphism

**Concepts**
- Virtual functions  
- Abstract classes  
- Interfaces  
- Deep vs shallow copy  

**Exercises**
- `Animal`
- `WrongAnimal`
- `Brain`
